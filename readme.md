# tubugl-3d-shape

3D shape(proceduaral box, procedural cube box) for [tubugl](https://github.com/kenjiSpecial/tubugl)

## install

To install it with npm:

```sh
npm install tubugl-3d-shape
```

To install it with yarn:

```sh
yarn add tubugl-3d-shape
```
